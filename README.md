## To Setup (Linux):

1. python3 -m venv env
2. source env/bin/activate
3. pip install -r requirements.txt
4. deactivate

## To Setup (Windows):

1. python -m venv env
2. env\Scripts\activate
3. pip install -r requirements.txt
4. deactivate

## To Run (Linux):

1. source run / bash run

## To Run (Windows):

1. ./run
